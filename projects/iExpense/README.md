## Requirements
* Xcode 13.2
* iOS 15.2

## Highlight
* Using `UserDefaults` to save simple data on the device.
* Using `JSONEncoder()` and `JSONDecoder()` to process data.
* Using `ObservableObject` and `@Published` to share data cross multiple SwiftUI view.
